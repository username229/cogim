Política de Privacidade da Cogim Galeria

**Última atualização: 3 de Dezembro de 2025**

A sua privacidade é importante para nós. Esta Política de Privacidade descreve como a Cogim recolhe, utiliza e protege as informações do utilizador ao aceder e utilizar a nossa Galeria.

1. Informação que Recolhemos

Uma vez que a Cogim Galeria é um portfólio estático, **não recolhemos ativamente informações pessoais** que identifiquem diretamente o utilizador, como nome, endereço de e-mail ou número de telefone.

No entanto, podemos recolher dados não pessoais através de ferramentas de análise:

  • **Dados de Uso:** Informações sobre como o Serviço é acedido e utilizado (ex: endereço IP anónimo, tipo de navegador, páginas visitadas, tempo gasto nessas páginas).
  • **Dados de Dispositivo:** Informações sobre o dispositivo usado para aceder ao Serviço (ex: modelo do dispositivo, sistema operativo).

2. Uso dos Dados

Os dados recolhidos (anónimos) são utilizados exclusivamente para os seguintes fins:

  • Para fornecer e manter o Serviço.
  • Para monitorizar o uso do Serviço (ex: quais categorias de fotos são mais populares).
  • Para detetar, prevenir e resolver problemas técnicos.
  • Para melhorar a experiência do utilizador e a funcionalidade da galeria.

3. Cookies e Tecnologias de Rastreio

Utilizamos Cookies e tecnologias de rastreio semelhantes para rastrear a atividade no nosso Serviço. Cookies são ficheiros com uma pequena quantidade de dados que podem incluir um identificador único anónimo.

A maioria dos navegadores permite que recuse a utilização de cookies. No entanto, se não aceitar cookies, poderá não conseguir utilizar algumas partes do nosso Serviço.

4. Segurança dos Dados

A segurança dos seus dados é importante para nós. Embora nenhum método de transmissão ou armazenamento eletrónico seja 100% seguro, esforçamo-nos para usar meios comercialmente aceitáveis para proteger os seus Dados de Uso.

5. Links para Outros Sites

A nossa galeria pode conter links para outros sites que não são operados por nós. Se clicar num link de terceiros, será direcionado para o site desse terceiro. Aconselhamos vivamente que reveja a Política de Privacidade de todos os sites que visitar.

6. Alterações a Esta Política de Privacidade

Podemos atualizar a nossa Política de Privacidade periodicamente. Iremos notificá-lo sobre quaisquer alterações publicando a nova Política de Privacidade nesta página.
